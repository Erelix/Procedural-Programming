Kodo standartai:

Kintamųjų pavadimai - angliški, logiški, trumpiniai gali būti naudojami tik funkcijose - main file pilni pavadinimai. Sudurtiniai žodžiai skiriami "_" (pvz., first_number).

Procedūrų pavadinimai - angliški, logiški, prasideda "my", sudurtiniai žodžiai neskiriami, tik jų pradžia rašoma iš didžiosios (pvz., myFillArray).

Pointerių pavadinimai - gale "_ptr" (pvz., result_file_ptr).



Iš https://github.com/MaJerle/c-code-style pasirinkti bei savo pridėti kodo rašymo standartai:

1. Tarpai po struktūrų pavadinimų
if (condition)
while (condition)
for (init; condition; step)
do {} while (condition)

2. Tarpai tarp elementų
a = 3 + 4;
for (a = 0; a < 5; ++a) 

3. Nėra tarpo tarp funkcijos pavadimio ir skliaustų ().
int32_t a = sum(4, 3);  

4. Defininimas to pačio kintamųjų tipo vieonoje eilutėje
char a, b; 

6. Pointerių inicializavimas į NULL.
char *string = NULL;

7. Po kiekvieno atminties išskyrimo tikrinimas ar ji buvo išskirta 
arr = ..malloc..
if (аrr != NULL)
	;

if (condition) {
	;
}
